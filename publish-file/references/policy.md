# Policy Warning

Показывай этот текст пользователю **перед каждым upload'ом** и жди явного согласия. Это guardrail от abuse, а не формальность. Без подтверждения дальше не иди.

---

Файл будет выложен по **публичной прямой HTTPS-ссылке**. Кто угодно, у кого есть эта ссылка, сможет скачать файл. Он будет доступен, пока активна твоя подписка CloudClaw.

Запрещено публиковать:

- нелегальный контент;
- пиратское ПО, торренты, любые нелегальные дистрибуции;
- malware;
- порнографию;
- любой контент, нарушающий применимое законодательство.

Нарушения караются ручным takedown и блокировкой аккаунта.
